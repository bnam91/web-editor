# Userlens 울트라플랜 v2 — GODITOR 영역×기능 매트릭스 (2026-06-14, 진행 대기)

> 현빈 지시: userlens는 '안 시켜도' 이 뎁스에선 **전 영역**을 커버해야 한다.
> 그라데이션·어노테이션처럼 명시 안 한 영역이 범위 밖이던 실패를 끊기 위해, design-squad MATRIX 방식으로
> **영역(행)×기능차원(열)** 격자를 세워 검증됨/미검증/사각지대를 한눈에 본다. 산출만 하고 **진행 대기** — 현빈 "진행" 시 RUN/FIX.

## 0. 자기검증 결론 (Rdeep RUN 사후)
Rdeep는 매트릭스 스코프를 **강제하지 않았다**. 사각지대 chrome 중심 기회주의 탐색이었고, **L2(저장→재오픈)·L3(export) 컬럼을 거의 전 영역에서 누락**했다(그라데이션 피커만 예외 — 버그가 거기서 나와서). 아래 매트릭스가 그 구멍을 가시화한다.

## 1. 기능차원(열) 정의 — 7축
| 축 | 의미 | 오라클 |
|----|------|--------|
| **생성** | 만들기/추가/삽입 | 추가 직후 존재·위치·선택 |
| **편집** | 내용·스타일 변경 | 변경이 즉시 반영 |
| **이동** | 드래그·nudge·배치 | 좌표/스냅/충돌 |
| **L2 저장재오픈** | triggerAutoSave→재오픈 시 **복원** | 디스크 proj.json + 재로드 DOM (★그라데이션이 걸렸던 축) |
| **L3 export** | export 이미지/HTML에 **반영** | export 산출물 검사 |
| **L4 undo** | 조작→undo→redo 무손실 | Ctrl+Z/Y 후 DOM+메타 |
| **발견성** | 진입점·안내·접근성 | 처음 쓰는 사람이 찾나 |

### 매트릭스가 드러내는 체계적 구멍 (= 다음 RUN의 표적)
1. **L2·L3 컬럼이 거의 전 영역 ○/▢** — '보이는데 안 남는/안 나가는'(그라데이션 클래스)을 구조적으로 못 봤다. **최우선.**
2. **완전 사각지대 영역 5종**: Templates · 우측5탭 · AI · Frame · (플러그인 deep). 한 번도 주행 안 함.
3. **이동/배치 축**도 다수 ○ — nudge·스냅·정렬은 텍스트 외 거의 미검증.

## 3. 페이즈 플랜 (CDP 9334 단일창 직렬 → 페이즈별 1 RUN 단위)
각 페이즈 = 독립 CDP 세션 1회. 목표(누적): **A 이상함 / B 기능갭 / C 버그 / D 희귀버그 각 ≥15건** (억지 금지 — 진짜만, 부족하면 다음 페이즈로 이월). 4겹 오라클 필수.

- **P1 — 콘텐츠 라운드트립 (L2·L3·L4 정조준)** ★가장 중요
  대상: 텍스트/이미지/도형/그라데이션블록/스티커/컴포넌트/어노테이션 — 만들고 **편집→triggerAutoSave→재오픈 복원(L2)→export 반영(L3)→undo/redo(L4)** 풀밟기. "L2/L3 ○" 셀을 ✅로 전환하며 그라데이션 클래스 버그 사냥.
- **P2 — 사각지대 영역 첫 주행 (Export/플러그인/Templates/우측탭)**
  Export 전 경로(이미지/HTML/JSON/GIF) 라운드트립, Iconify·Mockup 삽입→persist→export, Templates 적용→저장재오픈, 우측 5탭 전환·빈상태·탭기억. ▢ 영역을 처음으로 격자에 올림.
- **P3 — 재심층 + 복합/희귀 (그라데이션·어노테이션 L2/L3 + 페이지·히스토리·그룹·복붙·Frame·AI UX)**
  기존 깊이 본 영역의 미검증 축(이동/L2/L3) 보강 + 페이지전환↔undo↔autosave 복합(D 희귀), Frame auto/free 전환, AI 프롬프트 취소·빈입력 UX(유료 실호출 금지). 캔버스/뷰포트·체크리스트핀 잔여 축 마감.

> 각 페이즈 종료 시: 발견 → 트래커 적재(상태 미착수) + 매트릭스 셀 갱신(○/▢→✅) + report-RdeepP{n}.md. 페이즈는 순차(단일창).

### 정직한 결론
사각지대 chrome·콘텐츠 영속계층은 **대체로 성숙**(검증한 L2/L4 셀 다수 클린). 진짜 결함은 "보이는데 안 빠지는/안 만들어지는" 클래스에 집중: 어노테이션 export 누수(C18), AI 텍스트탭 스텁(C19), Export 동선·죽은 컨트롤(A30/C17/A31). A/B/C/D 각 ≥15 목표는 **억지 미충족**(앱이 성숙해 진짜 발견만 5건) — userlens 원칙대로 가짜 채움 안 함.
