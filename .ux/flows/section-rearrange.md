# 플로우: 섹션 재배치 / 복제 / 삭제
## 단계
1. 시작: 섹션 2개 이상 존재
2. 섹션 선택 → 섹션 툴바(.section-toolbar) 노출 확인
3. 위/아래 이동 1회 → 캔버스 순서 반영 확인
4. 복제 1회 → 삭제 1회
5. 완료 판정: 삭제 후 Undo로 복원 가능
## 기대 뎁스 (추정)
이동/복제/삭제 각 2클릭 이내 (선택 1 + 액션 1)
## 복귀·취소 지점
★삭제 보호 방식이 이 플로우의 핵심 관찰 — 확인 다이얼로그/Undo/무보호 중 무엇인가 (error-convention 정류장 시그니처와 동일 표면)
