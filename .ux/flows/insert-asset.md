# 플로우: 이미지 자산 삽입 (스크래치패드/업로드 → 블록)
## 단계
1. 시작: 에셋 패널 진입
2. 자산 선택 또는 업로드
3. 블록에 적용 (asset 블록의 빈 체크보드 = 자산 대기 신호 — 의도된 placeholder)
4. 완료 판정: 캔버스 블록에 이미지 표시
## 기대 뎁스 (추정)
3클릭 이내. AI 생성 경로는 별도 측정(생성 대기 시간은 feedback_delay 기준 비적용 — 진행 표시 존재 여부만)
## 복귀·취소 지점
