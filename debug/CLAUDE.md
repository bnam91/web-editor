# Debug 폴더 작업 지침

> 웹포트 CDP 디버깅 시 반복 실수 방지용 가이드

---

## 포트 맵

루트 `CLAUDE.md` → **작업 완료 후 QA 루틴** 섹션 참조

---

## 체크리스트

- [ ] 접속 전 `list_pages`로 현재 열린 포트 확인
- [ ] 별도 지시 없으면 9334(메인) 사용

---

## 기본 규칙

- **스크린샷 금지** — 포트 열어서 확인할 때는 반드시 `evaluate_script`(JS 실행) 또는 콘솔 로그로 확인
- DOM 상태 확인 → `evaluate_script`로 `innerHTML`, `getAttribute` 등 직접 조회
- 값·상태 확인 → `evaluate_script`로 변수·함수 반환값 콘솔 출력

---

## 자주 하는 실수

- 스크린샷으로 결과 확인하려 함 → evaluate_script / 콘솔로 대체할 것

### 스크린샷 허용 예외
- 레이아웃·렌더링 결과 (픽셀 위치, 겹침, 시각적 깨짐)
- CSS 애니메이션·트랜지션 상태
- 폰트 렌더링, 이미지 로딩 여부
